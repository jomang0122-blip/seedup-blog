## 전체 로드맵 (5단계)

### Step 1. 지금 바로 (조건 없음)
- [ ] 투자도구 3종(복리·절세·배당금 계산기) Blogger 게시 + 순환 URL 치환
- [ ] 쿠팡파트너스 가입 + 기존 시드업클래스 10편에 소급 삽입
- [ ] 공정위 제휴 고지문구 템플릿 준비
- [ ] 페이지 속도 개선(이미지 지연로딩, 안 쓰는 위젯 정리) — Core Web Vitals 개선, 구글 공식 사례 기준 광고수익 +10~18% 효과
- [ ] 테마 코드에 광고 넣을 자리 표시(문단 2·5 다음) — 실제 광고코드는 Step 2에서 삽입

### Step 2. 애드센스 심사 결과 확정 + 5개 자동발행 job이 4주 연속 무사고로 확인된 후
- [ ] 자동광고 + 앵커/비네트 광고 활성화
- [ ] Step 1에서 표시해둔 자리에 실제 광고코드 삽입
- [ ] 증권사 계좌개설 CPA 제휴 신청(네트워크 경유: 링크프라이스/애드픽) + 링크 삽입
- [ ] 시드업클래스 목차 허브 페이지 제작 + 전체 내부링크 구조화
- [ ] 구글 서치콘솔 월 1회 점검 루틴화

### Step 3. 3~6개월 후 (트래픽 안정화 확인 후)
- [ ] ETF 비교 시리즈 신설(예: "TIGER 200 vs KODEX 200")
- [ ] 절세계좌(IRP·연금저축) 제휴 추가
- [ ] 계산기 "결과 저장 시 이메일 입력" 게이트로 구독자 리스트 빌딩 시작

### Step 4. 조건 충족 시 (아래 3개 동시 충족)
- 트리거: 시드업클래스 30편 발행 + 이메일 리스트 500명 + 3개월 연속 일PV 500
- [ ] PDF 전자책 출시 (시드업클래스 초급~고급 재편집, 19,900~29,900원 1회 결제)
- [ ] 유료 뉴스레터 (Stibee, 월 9,900원 — 무료 시황과 겹치지 않는 심화 해석 콘텐츠)
- [ ] 프리미엄 계산기 애드온 (포트폴리오 리밸런싱 시뮬레이터 등)

### Step 5. 장기 검토 사안 (Step 1~4 안정화 후)
- [ ] 금융 관련 신규 주제(코인/부동산/원자재/채권 등) 매일 1건 추가 발행 확장안 — 구체 주제·우선순위 미정, 별도 논의 필요

## 핵심 참고사항 (에이전트 회의에서 확인된 것)

### 콘텐츠 우선순위 (3개 벤치마킹 에이전트가 독립적으로 동일 결론)
계산기/투자도구 > 시드업클래스 교육시리즈 > 시황뉴스(재방문 유도용으로 재포지션, 신규 트래픽 유입원 아님)

### 현실적 목표치
- 1년차 월 30~100만원이 검증된 현실적 벤치마크 ("월 1억" 등 강의업체 마케팅 주장은 불검증)
- 월 100만원 수익엔 일 PV 5,000 이상 필요, 트래픽 임계점은 포스팅 100개 이상
- 애드센스는 보조 수익, CPA·유료구독이 주력 수익원

### 반드시 지킬 레드라인
- **유사투자자문업법 리스크**: 유료로 특정 종목 매수·매도 반복 추천 시 3년 이하 징역 또는 1억원 이하 벌금 대상(2024.8 개정법). 계산기·교육·해설 중심 콘텐츠는 이 리스크 밖 — 앞으로도 "종목 추천"형 유료화는 하지 않는다.
- 애드센스는 금융 콘텐츠를 YMYL로 엄격 심사 — 자동생성 반복 콘텐츠가 "저가치 콘텐츠"로 반려될 위험, 검증기(validator.py) 등 차별화 장치 유지 필요
- 제휴 링크 고지문구는 글 앞부분(제목 또는 본문 최상단)에 필수, 특정사 편중 금지(제휴 삽입 글 비중 20~30% 이내)

### Blogger 플랫폼 제약
- 가능: 자동광고, 본문 내 수동 광고, 이미지 lazy loading, ads.txt
- 불가능: 서버단 최적화(CDN, 캐시 플러그인 등)
